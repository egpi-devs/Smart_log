QC Web App Deployment
======================
This folder contains everything needed to run the web application.

1. Ensure Python is installed on the server.
2. Install requirements by opening CMD in the 'backend' folder and running:
   pip install -r requirements.txt
3. Double-click the 'run_server.bat' file to start the server.
4. Users can access the app at: http://YOUR_SERVER_IP:8000
